# 测试数据生成器 - 方案 A：接口文档驱动

> 设计时间：2026-03-02 23:30  
> 核心理念：提供接口文档，自动生成测试数据

---

## 🎯 方案概述

**用户输入**：Swagger/OpenAPI 接口文档 URL 或文件  
**自动处理**：分析接口参数 → 推断数据库表 → 生成关联数据  
**输出**：完整的测试数据集（JSON/CSV/SQL）

---

## 📋 使用流程

### 步骤 1：提供接口文档

```bash
# 方式 1：Swagger URL
testdata-gen --swagger http://api.example.com/v2/api-docs

# 方式 2：本地文件
testdata-gen --openapi ./api-docs.json

# 方式 3：直接指定接口
testdata-gen --url http://api.example.com/api/v1/orders/create
```

### 步骤 2：选择生成策略

```bash
# 策略 1：单接口数据（只生成该接口需要的数据）
testdata-gen --strategy single

# 策略 2：完整流程数据（生成整个业务流程的数据）
testdata-gen --strategy workflow

# 策略 3：智能关联（自动分析关联表）
testdata-gen --strategy smart
```

### 步骤 3：配置生成参数

```bash
# 生成数据量
testdata-gen --count 10

# 数据格式
testdata-gen --format json  # 或 csv/sql

# 输出目录
testdata-gen --output ./test-data/
```

### 步骤 4：生成并验证

```bash
# 执行生成
testdata-gen generate

# 预览数据
testdata-gen preview

# 验证数据一致性
testdata-gen validate
```
