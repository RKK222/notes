#### 1. MiroFish - 群体智能引擎
**URL:** https://github.com/666ghj/MiroFish  
**Stars:** 30,285 (今日 +3,260)

**介绍:**
MiroFish 是一个简洁通用的群体智能预测引擎，由盛大孵化。项目将群体智能（Swarm Intelligence）理念通用化，可应用于各类预测任务。核心思想是模拟自然界群体行为（如鱼群、鸟群）的涌现智能，通过简单个体规则的交互产生复杂预测能力。

**技术架构:**
- Python 实现，模块化设计
- 群体行为规则引擎
- 预测结果融合机制
- 可视化分析工具

**创新点:**
- 将群体智能从特定领域（如优化）扩展到通用预测
- 简洁的 API 设计，易于集成
- 中文社区活跃，文档完善

**与现有技术对比:**
相比传统机器学习预测方法，MiroFish 不依赖大量训练数据，而是通过群体涌现行为进行预测。与集成学习不同，群体智能强调个体简单规则+交互涌现。

**局限:**
- 理论解释性待加强
- 超参数调优依赖经验

**发展潜力:**
群体智能在金融预测、天气预测、交通流预测等场景有广阔应用空间。项目今日星增长 3,260，显示社区高度关注。

#### 2. claude-mem - Claude Code 记忆插件
**URL:** https://github.com/thedotmack/claude-mem  
**Stars:** 36,964 (今日 +1,045)

**介绍:**
claude-mem 是 Claude Code 的插件，自动捕获编码会话全过程，使用 AI（Claude agent-sdk）压缩，并在未来会话中注入相关上下文。这是代理记忆（Agent Memory）方向的实用化实现，与 Trajectory-Informed Memory 论文和 Hindsight 项目形成呼应。

**技术架构:**
- TypeScript 开发，Claude Code 插件
- 会话捕获与存储
- AI 压缩与摘要生成
- 上下文检索与注入

**创新点:**
- 自动化：无需手动记录
- AI 压缩：保留关键信息，减少 token 消耗
- 情境感知：智能检索相关历史

**与 Hindsight 对比:**
Hindsight 是独立的代理记忆学习系统，claude-mem 专注 Claude Code 场景。两者都实现轨迹学习，但 claude-mem 更轻量、场景聚焦。

**发展潜力:**
随着 AI 编程助手普及，会话记忆成为刚需。claude-mem 展示了实用的记忆压缩和检索方案，值得跟踪。

### 学术论文（4 项）

#### 1. Structured Distillation for Personalized Agent Memory (arXiv:2603.13017)
**作者:** Sydney Lewis 等  
**机构:** Process Point Technologies

**介绍:**
论文提出结构化蒸馏方法，实现个性化代理记忆的 11 倍 token 压缩，同时保持检索质量。这是代理记忆效率优化的重要突破。

**技术方法:**
- 结构化表示学习
- 知识蒸馏压缩
- 个性化检索优化

**核心创新:**
- 11 倍压缩率，显著降低 token 成本
- 保持检索精度不下降
- 支持个性化记忆模式

**与 claude-mem 关系:**
论文提供理论基础，claude-mem 是实用实现。两者都解决代理记忆的存储和检索效率问题。

**应用价值:**
对于长期运行的 Agent 系统，记忆压缩是刚需。该方法可显著降低运营成本。

#### 2. ToolTree: Efficient LLM Agent Tool Planning via MCTS (arXiv:2603.12740)
**作者:** Shuo Yang 等  
**会议:** ICLR 2026

**介绍:**
ToolTree 提出基于双反馈蒙特卡洛树搜索（MCTS）和双向剪枝的 LLM 代理工具规划方法，显著提升工具调用效率。

**技术架构:**
- 双反馈 MCTS：正向执行反馈 + 反向价值反馈
- 双向剪枝：前向剪枝 + 后向剪枝
- 工具图表示

**核心创新:**
- MCTS 用于工具规划，探索效率高
- 双反馈机制加速收敛
- 双向剪枝减少搜索空间

**与现有技术对比:**
相比 ReAct、Plan-and-Execute 等线性规划方法，ToolTree 的树搜索能更好处理复杂工具组合。

**应用价值:**
对于需要多工具协作的 Agent 任务（如数据分析、代码生成），ToolTree 提供高效规划方案。

**作者:** I. de Zarzà, J. de Curtò, Jordi Cabot 等  
**会议:** AMSTA 2026 (Springer)

**介绍:**
论文提出代理 AI 系统的语义不变性验证框架，确保代理在不同情境下行为的一致性。这是 AI Testing 方向的形式化方法。

**技术方法:**
- 语义不变性定义
- 形式化验证框架
- 代理行为一致性检测

**核心创新:**
- 将形式化验证引入代理 AI
- 语义不变性作为验证属性
- 支持多情境测试

**与 ESAA-Security 关系:**
ESAA-Security 关注 AI 生成代码安全审计，本论文关注代理行为验证。两者都是 AI 安全的形式化方法。

**应用价值:**
对于安全关键场景（医疗、金融、自动驾驶），代理行为一致性验证是刚需。
