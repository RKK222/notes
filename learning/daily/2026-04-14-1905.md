## 技术扫描（30 项）

### AI Testing / Software Testing（10 项）

1. **HiL-Bench: Human-in-Loop Benchmark**
   - 来源：arXiv:2604.09408
   - URL：https://arxiv.org/abs/2604.09408
   - 介绍：编码代理判断力评估基准，测量"何时寻求帮助"的选择性升级技能。任务包含人类验证的阻塞器（信息缺失、模糊请求、矛盾信息），核心指标 Ask-F1 是问题精确率和阻塞器召回率的调和平均，架构上防止通过问题 spam 刷分。SWE 和 text-to-SQL 领域评估揭示普遍判断力差距。
   - 价值：首个代理判断力标准化评估，填补当前基准盲点

2. **Spatial-Gym: 空间推理逐步评估**
   - 来源：arXiv:2604.09338
   - URL：https://arxiv.org/abs/2604.09338
   - 介绍：Gymnasium 环境隔离空间约束推理，2D 网格谜题序列决策任务。8 个模型 3 种设置评估，最佳 GPT-OSS 120B 解决率 16.0%，远低于人类 98.0%。逐步格式帮助弱模型但损害强模型全局规划，视觉模型接收图像反而降低 73% 解决率。
   - 价值：空间推理诊断框架，揭示模型推理努力不随难度扩展的根本缺陷

3. **Hegel - 跨语言属性基测试协议**
   - 来源：Antithesis Blog
   - URL：https://antithesis.com/blog/2026/hegel/
   - 介绍：首个跨语言 PBT 标准，15 个语言实现，引入 APDL 属性描述语言。由 Hypothesis 创始人+Antithesis 联合开发。
   - 价值：解决多语言项目测试逻辑复用难题

4. **MutationLLM v2 - 语义感知变异测试**
   - 来源：arXiv:2604.09123
   - URL：https://arxiv.org/abs/2604.09123
   - 介绍：代码变更感知的 LLM 驱动变异测试，对比学习优化变异策略，等价变异体检测 96.8% 准确。
   - 价值：减少 67% 无效变异生成，缺陷检出率 +34%

5. **TestAgent-Pro - 端到端测试代理框架**
   - 来源：GitHub
   - URL：https://github.com/testagent-pro/testagent-pro
   - 介绍：分层多智能体架构，SWE-bench 82.3% 通过率。
   - 价值：自动化测试新范式

6. **c-CRAB - Code Review Agent Benchmark**
   - 来源：arXiv:2603.23448
   - URL：https://arxiv.org/abs/2603.23448
   - 介绍：首个代码审查代理基准，现有代理只解决 40% 任务。
   - 价值：标准化评估推动领域发展

7. **ReCodeAgent - 语言无关代码翻译**
   - 来源：arXiv:2604.07341

   - 介绍：多智能体代码翻译验证，6 语言 4 语言对，测试通过率 +60.8%。
