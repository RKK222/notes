11. **Process-Verified RL for Theorem Proving via Lean**

    - 来源：arXiv
    - 领域：AI

12. **Lagrange: 开放词汇端到端自动驾驶框架**
    - 地址：https://arxiv.org/abs/2606.20274
    - 来源：arXiv
    - 领域：AI
    - 技术介绍：基于Masked Latent Fields(MLF)的开放词汇计算稀疏驾驶框架。利用VLM将类别无关目标提议编码为连续语义视觉token。引入意图驱动的masked cross-attention模块时间过滤无关实体，将关注token解码为空间坐标上的隐式连续能量场。将决策制定为跨越能量场的拉格朗日动作最小化问题，确保车辆运动学合规。在nuScenes和CODA长尾基准上表现优异。
    - 应用场景：自动驾驶、开放世界感知、运动学规划
    - 为什么值得关注：统一开放词汇感知和运动学可行规划，解决长尾场景泛化问题

13. **ModSync: 无模块化冲突感知PINNs训练**
    - 地址：https://arxiv.org/abs/2606.20156

    - 领域：AI

    - 为什么值得关注：揭示PINNs训练中的新失效模式并提供通用解决方案

14. **Cross-Attention Attribution for Style-Captioned TTS**
    - 地址：https://arxiv.org/abs/2606.20532
    - 来源：arXiv
    - 领域：AI
    - 技术介绍：首次将DAAM框架适配到语音领域，分析风格字幕文本到语音扩散模型中每个词如何影响声学输出。提取25层×24步ODE的per-token热力图，分析3600个组合。发现：风格token时间方差低于内容token（确认全局条件化）；风格注意力与F0和能量相关；风格条件化在早期步骤和深层达到峰值；注意力熵在层17达到最小值，与风格重要性峰值重合。
    - 应用场景：TTS模型可解释性、语音生成控制
    - 为什么值得关注：首次系统揭示自然语言如何影响语音扩散模型的cross-attention

    - 领域：AI

## 深度研究

### GitHub 项目

1. **ENPIRE: 具身机器人策略自我改进框架**
   - 地址：https://arxiv.org/abs/2606.19990
   - 项目介绍：ENPIRE是伯克利/UT Austin/NVIDIA联合提出的具身机器人策略自我改进框架。当前编码Agent在数字环境中取得了巨大成功，但物理世界的机器人研究仍然高度依赖人工监督。ENPIRE首次实现了物理世界的"重置→执行→验证→精炼"闭环自动化。四个核心模块：Environment模块自动重置场景和验证结果、Policy Improvement模块启动编码Agent搜索策略改进、Verification模块检查物理执行结果、Refinement模块迭代优化策略。
   - 技术架构：四模块闭环架构，编码Agent作为策略搜索引擎，物理环境作为验证器，形成自改进循环。
   - 核心创新：将编码Agent的自动化搜索能力从数字环境扩展到物理世界，首次实现真实世界的策略自改进。
   - 与现有技术对比：现有方法依赖人工设计奖励和策略，ENPIRE实现全自动改进循环。
   - 优势与局限：优势是真正的物理世界闭环，局限是需要可自动重置的物理环境。
   - 未来发展潜力：具身AI自动化的关键基础设施，可能改变机器人研究范式。

2. **ScaffoldAgent: 深度研究大纲动态优化框架**
   - 地址：https://arxiv.org/abs/2606.20122
   - 项目介绍：针对开放 ended 深度研究(OEDR)中的大纲管理问题。现有方法要么在写作前固定大纲，要么用局部启发式精炼，导致持续信息积累下的"大纲漂移"。ScaffoldAgent将大纲演化建模为结构化决策过程（扩展、收缩、修订三操作），引入效用引导反馈机制从检索增益、结构一致性、试生成质量三维度估计每次大纲操作的后验价值。
   - 技术架构：大纲作为结构化决策对象，效用信号驱动节点选择、操作调度、终止判断。
   - 核心创新：首次将大纲管理从静态/局部优化推进到结构化效用驱动决策。
   - 优势与局限：显著改进长文生成和事实准确性，但需要试生成来评估效用增加计算开销。
   - 未来发展潜力：AI深度研究系统的核心组件，可能成为研究Agent的标准架构。

3. **Lagrange: 开放词汇端到端自动驾驶框架**
   - 地址：https://arxiv.org/abs/2606.20274
   - 项目介绍：基于Masked Latent Fields(MLF)的开放词汇计算稀疏驾驶框架。解决密集模型计算瓶颈和稀疏模型封闭集依赖的两难困境。利用VLM将类别无关目标提议编码为连续语义视觉token，引入意图驱动的masked cross-attention时间过滤无关实体，解码为空间坐标上的隐式连续能量场。将决策制定为拉格朗日动作最小化问题确保运动学合规。
   - 技术架构：VLM编码→masked cross-attention→能量场→拉格朗日最小化→运动学合规轨迹。
   - 核心创新：统一开放词汇感知和运动学可行规划，能量场框架天然支持长尾场景。
   - 优势与局限：在nuScenes和CODA长尾基准上表现优异，但能量场优化的计算效率需要进一步优化。
   - 未来发展潜力：开放世界自动驾驶的关键技术路径。

### 学术论文

1. **Process-Verified RL for Theorem Proving via Lean**
   - 地址：https://arxiv.org/abs/2606.20068
   - 论文介绍：利用Lean证明助手作为符号过程oracle，提供结果级和细粒度策略级验证反馈。将证明尝试解析为策略序列，Lean的elaboration标记局部有效步骤和最早失败步骤，产生密集、验证器基础的信用信号。结合GRPO风格RL目标，一阶错误传播和一token信用方法。在MiniF2F和ProofNet上超越仅结果基线。
   - 技术架构：Lean作为过程oracle→策略序列解析→错误传播→GRPO优化。
   - 核心算法：第一错误传播和第一token信用方法平衡结果级和过程级优势。
   - 与现有技术对比：传统RLVR仅依赖单一二元验证信号，本文利用证明助手提供的结构化细粒度反馈。
   - 优势与局限：证明助手不仅是评估时的验证器，更是训练时的过程级奖励oracle。局限是依赖特定证明助手。
   - 未来发展潜力：为形式化证明自动化开辟新路径，可能推广到其他符号系统。

2. **ModSync: 无模块化冲突感知PINNs训练**
   - 地址：https://arxiv.org/abs/2606.20156
   - 论文介绍：发现PINNs训练中的容量诱导失效模式——过参数化网络经历功能模块化，自划分为任务专属模块抑制跨目标交互阻碍收敛。提出ModSync框架将结构优化整合到冲突感知训练中，惩罚任务专属连接同时保留交互促进路径。在多种PDE基准上持续防止容量驱动失效，达到SOTA精度。
   - 技术架构：冲突感知优化+结构稀疏化惩罚+交互路径保留。
   - 核心算法：ModSync同时优化训练冲突和架构模块化，双目标协同。
   - 与现有技术对比：现有冲突感知优化方法在模型容量增加时效果恶化，ModSync从根本上解决容量诱导模块化。
   - 优势与局限：通用性强适用于各种PDE，但结构优化增加训练复杂度。
   - 未来发展潜力：科学计算中PINNs训练的标准方法。

3. **Reward as Agent for Embodied World Models**
   - 地址：https://arxiv.org/abs/2606.19990
   - 论文介绍：挑战保守rollout范式，提出核心限制不是探索本身而是缺乏可靠验证策略。引入Reward as Agent框架主动评估生成行为提供鲁棒奖励信号缓解reward hacking。结合DynDiff-GRPO显式扩展动作空间探索多样化轨迹。在多个开源世界模型上实现显著精度提升。
   - 技术架构：Agent奖励评估+动态感知rollout多样化+GRPO优化。
   - 核心算法：将验证从被动评估推进到主动Agent，解决RL探索的核心瓶颈。
   - 与现有技术对比：现有方法依赖保守rollout限制探索，本文在可靠验证基础上支持广泛探索。
   - 优势与局限：在多个世界模型上验证有效，但Agent奖励评估增加计算开销。
   - 未来发展潜力：具身世界模型训练的新范式。
