## 深度研究

### GitHub 项目
1. **MetaResearcher** - 深度研究Agent训练框架
   - 问题: 现有深度研究训练受限于静态环境、事实检索任务设计、结果级RL低效
   - 方法: 四维度协同——演化虚拟世界(时序动态+对抗虚假信息)、发现导向任务(假设生成+矛盾解决)、自反思Meta-Reward(GRPO框架)、异构多Agent Swarm(Scout/Filter/Synthesizer)
   - 创新: 零边际API成本训练，目标GAIA/Xbench-DS性能提升+对抗条件下认知鲁棒性
   - 价值: 解决深度研究Agent训练数据稀缺和评估静态化问题

2. **Multi-Agent Transactive Memory (MATM)** - 群体级Agent经验共享
   - 问题: Agent轨迹通常单次使用后丢弃，新Agent重复发现已有解决方案
   - 方法: 生产者Agent贡献轨迹到共享仓库，消费者Agent检索提升任务执行，ALFWorld/WebArena验证
   - 创新: 将RAG从人类文档扩展到Agent轨迹，无需协调或联合训练
   - 价值: 开放Agent生态系统的群体经验共享设计模式

3. **ORAgentBench** - 端到端运筹学Agent评估
   - 问题: 现有OR评估分离建模与求解，依赖预形式化实例
   - 方法: 107人工审查任务，隔离环境+自然语言简报+多文件数据+隐藏验证器
   - 创新: 首次端到端评估，覆盖schema有效性+硬约束可行性+目标质量
   - 价值: 揭示Agent从可行代码到可靠决策的差距

### 学术论文
1. **ICT: Independent Combinatorial Tokens for RLVR** (arXiv:2606.19771)
   - 问题: RLVR均匀Token更新导致熵崩溃或熵爆炸
   - 方法: JS散度识别分布模式独特的Token作为关键分支点，仅更新Top 10%独特Token
   - 创新: 从标量不确定性转向Token logit分布属性优化，Shannon+Rényi双重熵理论保证
   - 价值: Qwen2.5 0.5B/1.5B/7B平均+4.58% pass@4，最大+14.9%

2. **SEVRA: Selective Verification for Budget-Aware Reasoning** (arXiv:2606.19808)
   - 问题: 额外推理不是均匀有价值的——修复失败/浪费计算/引入有害改变
   - 方法: 部署层控制器决定保留冻结求解器答案或激活验证，可恢复性感知门控
   - 创新: 将推理验证视为部署分配问题而非新验证器问题
   - 价值: Math5 76.3%准确率+26.8% token减少+有害翻转2.2%→1.0%

3. **Black-Box UE Systematic Evaluation** (arXiv:2606.19868)
   - 问题: 黑盒不确定性估计方法碎片化，缺乏统一比较
   - 方法: 五分类(言语化/采样/解释/多Agent/混合)，24方法×4模型×4数据集统一基准
   - 创新: 首个系统化黑盒UE评估框架，答案空间比较方法最有效
   - 价值: 为可信LLM部署提供不确定性估计实用指南

## 新领域收获
1. **Autotelic AI哲学**: 自主目标生成Agent的核心问题不是如何生成目标，而是如何生成和相对化目标所归属的"自我"——嵌入性使个体化非唯一，同一动力学允许多个有效划分
2. **群体级Agent记忆(MATM)**: 将RAG从人类文档扩展到Agent轨迹，生产者-消费者模式实现开放生态系统的经验共享
3. **端到端运筹学Agent评估(ORAgentBench)**: 揭示Agent从"写出可行优化代码"到"做出可靠运营决策"之间的巨大差距
