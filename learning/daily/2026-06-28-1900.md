## 技术扫描（30+）

### AI Testing / Software Testing（10 个）

1. **OpenRCA 2.0: From Outcome Labels to Causal Process Supervision** (arXiv:2606.27154) - 首个跨系统根因分析基准，500 实例带逐步因果标注。发现 11 个前沿 LLM 平均仅 20.7% 恢复精确根因集合。关键发现：76% 代理识别至少一个正确根因，但仅 61.5% 能将其锚定到已验证的因果传播路径。"未扎根诊断"问题被结果级评估隐藏。PAVE 逐步标注协议利用故障注入已知干预重建因果传播路径。对 Agent 可观测性和故障诊断测试有直接价值。

2. **Semantic Early-Stopping for Iterative LLM Agent Loops** (arXiv:2606.27009) - 研究多 Agent 循环（Writer+Critic）的语义早停机制。证明确定性终止和良定义性（机器验证）。在 HotpotQA 60 题测试集上，无裁判语义停止器在质量不变（Delta-IS=-0.004, p=0.81）下减少 38% 操作 token。发现"哪轮最优"比"何时停止"更难。对 Agent 执行成本优化有直接价值。

3. **BINEVAL: Binary Questions for Interpretable LLM Evaluation and Self-Improvement** (arXiv:2606.27226, ICML 2026 Workshop) - 将评估标准分解为原子二元问题，聚合为可解释多维分数。在 SummEval、Topical-Chat、QAGS 上匹配或超越 UniEval 和 G-Eval。关键优势：避免 LLM Judge 天花板效应，更好区分边界和明显缺陷输出。同一问题级反馈支持迭代提示优化。ICML 2026 Workshop 录用。

4. **Diagnosing Task Insensitivity in Language Agents** (arXiv:2606.26918) - 发现 Agent OOD 泛化弱的核心原因是"任务不敏感"：面对相似但不同的任务，模型应用训练模式而非解决当前任务。即使指令语义损坏，模型仍继续执行原始任务对齐的动作。训练时注意力从任务 token 漂移到局部观察。提出 Task-Perturbed NLL Optimization 对比正则器，改善任务敏感性和 OOD 泛化。

5. **Where Do CoT Training Gains Land in LLM-based Agents?** (arXiv:2606.26935) - 发现 CoT 训练主要改善的是"提示动作"质量而非 CoT 推理本身。后续 checkpoint 更不可能根据 CoT 修改动作，表明更多依赖提示。选择性屏蔽动作 token 监督改善域外泛化。挑战"CoT 训练提升推理能力"的直觉。

6. **Autoformalization of Agent Instructions into Policy-as-Code** (arXiv:2606.26649, AIWILD@ICML 2026) - 将 Agent 提示、MCP 工具描述和自然语言策略文档自动形式化为 Cedar Policy Language 的形式化验证策略。LLM 生成器-批评器循环。在 MedAgentBench 上覆盖比手工编码符号执行更多的源自然语言规范。对 Agent 安全治理有直接价值。

7. **LeanGuard: A Fast and Light Approach for Robust Moderation** (arXiv:2606.26686) - 挑战"安全护栏需要 CoT 推理"的假设。395M 标签-only 编码器达到 F1 82.90，匹配更大解码器推理护栏，但推理计算减少约 100 倍。在训练标签噪声下保持鲁棒性，在严格误报率下保留更多召回。发现当前护栏基准可能不够难以奖励推理。

8. **Distilling and Compiling Agent Traces into Reusable Procedural Skills (SkillDisCo)** (arXiv:2606.26669) - 在 FSM 定义场景中，将成功轨迹蒸馏为可复用参数化控制流子图，编译为可调用的、可执行的、可验证的程序技能。在 ALFWorld 和 WebArena 上提高成功率并减少 Agent 轮次。对 Agent 技能工程有直接价值。

9. **Prompt Injection in Automated Résumé Screening** (arXiv:2606.27287, ACL 2026 Findings) - 研究简历筛选中 LLM 的提示注入攻击。当简历质量同质且少数候选人注入时，注入可靠改善排名。但当注入普遍化时效果迅速崩溃。异构质量下注入平均效果较低，但偶尔允许低质量候选人超越高质量候选人。

10. **Vulnerability of Natural Language Classifiers to Evolutionary Generated Adversarial Text (GAversary)** (arXiv:2606.27215) - 混合遗传算法生成对抗攻击，使用 GloVe 嵌入提出词替换。黑盒攻击，仅需 logit 输出。在最佳情况下将目标模型准确率从 76.8% 降至 5.8%（BAE 降至 27.6%）。对 NLP 模型安全测试有价值。

### AI / LLM 技术（11 个）

1. **The Capability Frontier: Benchmarks Miss 82% of Model Performance** (arXiv:2606.26836) - 引入 Capability Frontier：Pareto 前沿刻画在模型和生成间最优选择下每个成本水平的最佳可达性能。研究 21 个 LLM×16 基准，纠正单模型评估偏差减少 54% 错误率，额外纠正单次运行偏差进一步改善。关键发现：不同模型根据专业化获得不同题目正确，多次生成可选择性保留。评估方法论范式转变。

2. **When Does Combining Language Models Help? A Co-Failure Ceiling** (arXiv:2606.27288) - 67 模型×21 提供商验证：任何输出为成员模型答案之一的策略，准确率不能超过 1-β（β 为所有模型在同一查询上全部错误的比率）。在开放数学上观测 β=0.052 vs 高斯 copula 预测 0.023（低估 2.5 倍）。在可检查任务上，组合模型很少超越单个最佳模型。增益来自模型在不同问题上失败，而非添加更多模型。

3. **Memory Depth, Not Memory Access: Selective Parametric Consolidation (EVAF)** (arXiv:2606.26806) - 研究记忆深度（持久的目标条件倾向写入小参数存储）与检索访问的互补性。引入 loop-drift 协议控制压力测试。EVAF 惊喜-效价门控 LoRA 巩固机制，在目标持久性和卸载后恢复上最强（0.812-0.904），每 200 事件仅 2-3 次参数写入。选择性和执行性分解为两个可控维度。

4. **Joint Learning of Experiential Rules and Policies (JERP)** (arXiv:2606.27136) - 同时更新长期经验规则池和策略。决策时检索任务相关规则并条件化 Agent。每轮后使用收集轨迹优化策略并修订规则池（比较当前 rollout 与参考成功轨迹）。在 AlfWorld 和 WebShop 上对复杂交互任务产生一致的决策性能增益。

5. **Einstein World Models** (arXiv:2606.26969) - LLM 推理系统中嵌入视觉-时间展开的新蓝图。LLM 调用世界模块（非世界模型）生成短场景展开，返回的展开作为可检查假设支持后续推理。将 LLM 工具调用能力扩展到视觉思维实验领域。挑战"复杂思维只能通过语言捕获"的假设。

6. **TOPS: First-Principles Visual Token Pruning** (arXiv:2606.27161) - 从第一性原理重新审视视觉 token 剪枝，构建 Token Optimal Preservation Sets。通过自顶向下信息论分析识别三个基本原则：任务相关性、信息覆盖、语义多样性。无训练、模型无关剪枝模块，可应用于各种 MLLM。在 7 个 MLLM 基座上广泛实验。

7. **ReTeX: Learning to Recover Task Experts from Multi-Task Merged Model** (arXiv:2606.26902) - 将参数干扰建模为仿射变换（可近似为加性偏移），预测这些偏移以撤销参数干扰并从单个合并检查点恢复任务专家性能。引入无路由器任务标识符（基于 SVD 子空间签名）。在视觉和 NLP 领域恢复 95%+ 个体专家性能，同时显著改善未见任务泛化。

8. **Process Harness for Uplifting Legacy Workflows to Agentic BPM** (arXiv:2606.27188) - 在确定性工作流引擎周围放置策略治理代理层，拦截指定控制点贡献推理、适应和监督。TDF 模型分解为 TaskAgent（知识密集型执行）、DecisionAgent（网关路由）、FlowAgent（运行时流适应）。在贷款审批工作流上验证。独特地调和了命令式要求（确定性执行）和规范要求（代理自主性）。

9. **EGG: Expert-Guided Agent Framework for Kernel Generation** (arXiv:2606.26758) - 将专家优化原则纳入 LLM 决策，分两阶段：(1) 算法结构设计 (2) 硬件特定调优。阶段感知多 Agent 协作机制。在 KernelBench 上达到 PyTorch 2.13x 加速，超越现有 Agent 和 RL 方法。对 AI 基础设施性能优化有价值。

10. **NebulaExp: Empirical Post-Training Pipeline via Full-Scale Ablation** (arXiv:2606.26671) - 完全透明的消融驱动后训练管道，基于 Qwen3-8B-base。3.84M SFT 样本 + 200K RL 候选池。Instruct 分支三阶段 SFT 从 55.01 提升至 60.99，GRPO RL 进一步至 61.85。MOPD 融合四位领域专家教师仅 10K 样本提升 4.18 分。为 8B 级 LLM 提供完全可复现后训练配方。

11. **Knowledge-base Augmented Retrieval for Language Models** (arXiv:2606.26807) - 训练模型产生特殊 token 触发知识库查询，实现：(1) 事实知识无需重新训练即可更新 (2) 输出事实可追溯到知识库 (3) 小模型达到与大模型相同的事实准确性。在短和长格式生成中改善事实扎根。

### 新领域探索（12 个）

1. **AHOIS: Socratic Agents for Autonomous Scientific Discovery** (arXiv:2606.26722) - 多 Agent AI 科学家将苏格拉底助产术嵌入闭环实验。物理批评 Agent 通过因果提问、约束检查、反例生成和证伪标准制定审查假设。在真实多模光纤光学平台上验证，自主提出并验证随机干扰编码假设。分类准确率 MNIST 76.97%、Fashion-MNIST 83.17%。从工作流自动化到认知自主的路线。

2. **Scientific Discovery as Meta-Optimization** (arXiv:2606.26728) - 将研究形式化为元优化，同时优化问题和评估标准。"共识目标聚合"：LLM 生成的目标函数通过相关性加权投票组合。在 3-SAT 数字 MemComputing 算法发现上，将基线缩放从 N^2.51 降至 N^1.33，最大实例 67x 加速。

3. **LithoDreamer: Physics-Informed World Model for Computational Lithography** (arXiv:2606.26713, ICML 2026) - 首个计算光刻物理信息世界模型框架。将"Layout-Mask-Resist Image-ADI"管道建模为决策驱动多步演化系统。对比变分优化范式对比干预路径的潜差异。ICML 2026 录用。对半导体制造 AI 有直接价值。

4. **EO-WM: Physically Informed World Model for Earth Observation** (arXiv:2606.27277) - 视频扩散 Transformer 用于多光谱 EO 预测。物理信息条件框架通过气候基线、天气异常和累积物理压力信号表示气象强迫。极端夏季基准和季节匹配对基准测试天气响应行为。NDVI 下降幅度误差相对降低 5.63%。

5. **Language-Based Digital Twins for Elderly Cognitive Assistance** (arXiv:2606.27334, PETRA 2026) - 利用 LLM 模拟老年人对话行为的语言数字孪生框架。多头条件变分自编码器联合测量重建质量和预测认知评分。在 I-CONECT 数据集上保留身份特定特征，MoCA 预测误差与真实数据相当。

6. **Narrative-Grounded World Visual Attention in Dynamic 3D Story Worlds** (arXiv:2606.26964) - 动态 3D 故事世界中的摄像机规划。"先看再动"框架分离观察规范与运动执行。语义观察契约+蒙特卡洛视点搜索+语义轨迹接地。50 故事×457 场景×1585 镜头基准。

7. **ResilPhase: Noise-Resilient Macro-Trajectory Extrapolation for Diffusion** (arXiv:2606.26769, ECCV 2026) - 将加速推理重新公式化为 ODE 空间中的稳定宏观轨迹外推。无导数重心拉格朗日外推器绕过导数不稳定性。有界相位映射正则化外推域。在 FLUX.1-dev 和 HunyuanVideo 上达到 SOTA 保真度。ECCV 2026 录用。

8. **Kalman Prototypical Networks for Few-shot Fault Detection** (arXiv:2606.26710) - 度量型少样本学习框架用于燃气轮机故障诊断。将类原型演化建模为潜随机状态。Modelica 海上 CCGT 动态仿真生成合成数据集。在准确性和稳定性上超越 Matching Networks、Relation Networks 和 MAML。

9. **Latent ODE for Spatiotemporal Modeling of Cine Cardiac MRI** (arXiv:2606.26718) - 72,386 UK Biobank 参与者研究。心脏速率感知神经 ODE 动力学+图网格自编码器重建解剖一致 3D+t 心室运动。Cox 比例风险模型测试偏差是否预测心力衰竭。C-index 从 0.704 提升至 0.785。

10. **Risk-Conditioned Causal Grounding for Hallucination-Resistant Report Generation (TAVR-VLM)** (arXiv:2606.26874) - 风险条件因果接地注意力（R-CGA）实例化"风险→区域→词"结构接地路径。在 1,482 患者 M³TAVR 队列上 AUROC 0.896，幻觉率降至 8.1%。

11. **Generative Retrieval via Diffusion Transformer (MO-DiT+HPPO)** (arXiv:2606.26899) - 模式保持属性检索：给定种子集生成同时满足目标属性并保持模式的项目。多域度量有序延续预训练+尾部质心微调+混合策略偏好优化。

12. **Simulation-based Inference for Epidemiological Models** (arXiv:2606.26886) - 神经后验估计作为 MCMC 可扩展替代方案。在 COVID-19 ICU 占用数据上，SBI 在单 GPU 上 60-70 秒达到 MCMC 1000 秒的相当后验性能。201 天推理问题 SBI 157 秒 vs MCMC 19,000 秒。

## 深度研究

### GitHub 项目

#### 1. Capability Frontier — 基准遗漏 82% 模型性能
- **地址**: https://arxiv.org/abs/2606.26836
- **类型**: 论文
- **领域**: LLM Evaluation / 评估方法论
- **技术介绍**: Capability Frontier 提出一个根本性的评估范式转变。现有基准报告单个模型在单次运行中的准确率，系统性地低估了 LLM 的真实能力。两个关键偏差来源：(1) 不同模型根据专业化获得不同题目正确 (2) 给定预算下可采样多次生成并选择性保留。Capability Frontier 是 Pareto 前沿，刻画在模型和生成间最优选择下每个成本水平的最佳可达性能。研究 21 个 LLM×16 基准（编码、推理、医学、事实性、指令遵循、Agent 任务），纠正单模型评估偏差减少 54% 错误率，额外纠正单次运行偏差进一步改善。
- **应用场景**: LLM 基准测试、模型能力评估、成本-性能优化
- **为什么值得关注**: 直接挑战当前 LLM 评估的基础假设。如果基准确实遗漏了 82% 的模型性能，那么大量关于"模型能力"的结论都需要重新审视。对 AI 测试和评估基础设施有根本影响。

#### 2. Co-Failure Ceiling — 多模型组合理论上限
- **地址**: https://arxiv.org/abs/2606.27288
- **类型**: 论文
- **领域**: LLM Evaluation / 多模型系统
- **技术介绍**: 对 67 模型×21 提供商的大规模实证研究，形式化多模型组合（路由、投票、级联、融合、MoA）的理论上限。关键发现：任何输出为成员模型答案之一的策略，准确率不能超过 1-β，其中 β 是所有模型在同一查询上全部错误的比率。传统诊断（平均成对错误相关性 ρ）无法识别 β。在开放数学上观测 β=0.052 vs 高斯 copula 预测 0.023（低估 2.5 倍）。在可检查任务上，组合模型很少超越单个最佳模型。
- **应用场景**: 多模型系统集成、路由策略设计、评估上限预测
- **为什么值得关注**: 为多模型组合系统提供了首个可计算的天花板。在训练路由器之前即可预测最大增益。对 Agent 编排和多模型推理基础设施有直接价值。

#### 3. AHOIS — 苏格拉底式自主科学发现
- **地址**: https://arxiv.org/abs/2606.26722
- **类型**: 论文
- **领域**: 自主科学发现 / 多 Agent 系统
- **技术介绍**: AHOIS 将苏格拉底助产术嵌入闭环实验。物理批评 Agent 通过因果提问、约束检查、反例生成和证伪标准制定审查假设。在真实多模光纤光学平台上验证（高维系统，复杂波变换、间接检测、环境漂移）。无先验编码方案、分类器或散斑模型，系统自主提出并验证随机干扰编码假设，发现任务自适应稀疏测量策略，诊断不同失败模式。分类准确率 MNIST 76.97%、Fashion-MNIST 83.17%。消融显示苏格拉底提问改善物理一致性、假设完整性、不确定性校准和实验计划有效性。
- **应用场景**: 自主科学研究、物理系统探索、实验设计
- **为什么值得关注**: 从工作流自动化到认知自主的里程碑。大多数 AI 科学家系统仍是程序化的（执行人类设计师固定的工作流），AHOIS 展示了真正的认识自主——构建、挑战和修正物理解释的能力。

### 学术论文

#### 1. BINEVAL — 可解释二元评估框架
- **地址**: https://arxiv.org/abs/2606.27226
- **类型**: 论文（ICML 2026 Workshop）
- **领域**: LLM Evaluation / 可解释评估
- **技术介绍**: BINEVAL 将评估标准分解为原子二元问题，聚合为可解释多维分数。给定任务提示，元提示生成细粒度评估问题，LLM 独立回答每个输出，产生透明的问题级反馈和校准总体分数。在 SummEval、Topical-Chat、QAGS 上匹配或超越 UniEval 和 G-Eval，在事实一致性基准 QAGS 上表现尤其强。关键优势：避免 LLM Judge 天花板效应，更好区分边界和明显缺陷输出。同一问题级反馈支持迭代提示优化。
- **应用场景**: LLM 输出评估、提示优化、多轮评估
- **为什么值得关注**: 解决了 LLM 评估的核心痛点—— holistic LLM Judge 产生不透明分数难以调试。BINEVAL 提供任务无关、无训练、可解释的评估框架，兼具强实证性能和实用诊断价值。

#### 2. OpenRCA 2.0 — 因果过程监督
- **地址**: https://arxiv.org/abs/2606.27154
- **类型**: 论文
- **领域**: Agent 测试 / 根因分析
- **技术介绍**: 首个跨系统 RCA 基准，500 实例带逐步因果标注。PAVE 逐步标注协议利用故障注入已知干预重建因果传播路径。关键发现：11 个前沿 LLM 平均仅 20.7% 恢复精确根因集合。但"未扎根诊断"问题更严重：76% 代理识别至少一个正确根因服务，但仅 61.5% 能将其锚定到已验证因果传播路径。结果级评估隐藏此失败模式。
- **应用场景**: Agent 故障诊断、可观测性测试、系统可靠性
- **为什么值得关注**: 揭示了 Agent 根因分析的核心问题不是"找到根因"而是"验证因果路径"。对 AI 运维和 Agent 可观测性测试基础设施有直接价值。

#### 3. EVAF — 记忆深度 vs 检索访问
- **地址**: https://arxiv.org/abs/2606.26806
- **类型**: 论文
- **领域**: Agent 记忆 / 参数化巩固
- **技术介绍**: 研究"记忆深度"（持久的目标条件倾向写入小参数存储）与检索访问的互补性。引入 loop-drift 协议控制压力测试（检索索引保持完整但工作上下文卸载）。EVAF 惊喜-效价门控 LoRA 巩固机制，在目标持久性和卸载后恢复上最强（0.812-0.904），每 200 事件仅 2-3 次参数写入。机制控制显示选择性巩固分解为选择和执行两个可控维度。
- **应用场景**: 长期 Agent 记忆、目标持久性、参数化知识存储
- **为什么值得关注**: 区分了"记忆访问"（检索）和"记忆深度"（参数化巩固）两个正交维度。对长期运行的 Agent 系统有根本影响——Agent 不仅需要记住事实，还需要记住行为倾向。

## 技术趋势
1. **LLM 评估方法论重构**: Capability Frontier 和 Co-Failure Ceiling 两篇论文共同指向一个趋势：现有基准系统性地低估或误估 LLM 能力。Pareto 前沿和全错比率 β 天花板提供了可计算的能力边界，BINEVAL 提供了可解释的评估框架。评估从"单次运行准确率"转向"成本-性能 Pareto 前沿"。
2. **Agent 记忆范式从检索到巩固**: EVAF 和 JERP 两篇论文共同指向一个趋势：Agent 记忆不再只是检索过去事实，而是通过参数化巩固将经验写入模型本身。记忆深度（目标持久性）和检索访问（事实召回）是两个正交维度。
3. **自主科学发现从工作流到认知**: AHOIS 和 Meta-Optimization 两篇论文共同指向一个趋势：AI 科学发现从执行人类设计的工作流转向真正的认识自主——构建、挑战和修正物理解释，同时优化问题和评估标准。

## 新领域收获
1. **Capability Frontier 评估方法论**: Pareto 前沿刻画每个成本水平的最佳可达性能，基准遗漏 82% 模型性能。对 AI 测试基础设施有根本影响——评估不再只是"单次运行准确率"。
2. **Co-Failure Ceiling 理论**: 67 模型×21 提供商验证全错比率 β 天花板。多模型组合增益来自模型在不同问题上失败，而非添加更多模型。为 Agent 编排提供理论上限。
3. **Process Harness for Agentic BPM**: 确定性工作流引擎 + 策略治理代理层。独特地调和命令式要求（确定性执行）和规范要求（代理自主性）。对企业级 Agent 部署有直接价值。
