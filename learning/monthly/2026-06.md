## 本月技术趋势（前 4 日观察）

1. **Agent 评估基础设施化**：GLIDE（PPI 估计器统一库）和 TraceGraph（轨迹决策图谱）同时出现，标志着 Agent 评估从"手工跑基准"走向"标准化基础设施"
2. **专家知识蒸馏→Agent 技能包**：COLLEAGUE.SKILL（18.5K stars）代表"将人类 expertise 编码为 Agent 技能"的新范式
3. **形式化验证 × RL 融合加速**：Dafny RLVR、Value Function 即 Supermartingale 证书、ASP-based RL 抽象
4. **JEPA 架构扩展**：CHARM 将 JEPA 从视觉扩展到时间序列，ICML 2026
5. **Agent 推理结构显式化**：LinTree 揭示"搜索历史本身不够，必须显式表示树结构"
6. **凭证检测从二元到三分类**：CNN-CodeBERT（ICSME 2026）显著降低误报
7. **RLVR 小模型代码生成**：Qwen3-0.6B/Llama3.2-1B + RLVR pass@1 +13pp
8. **AI 基础设施投资加速**：Alphabet $80B 融资 + OpenAI on AWS
9. **代理效率优化三维度**：headroom（内容压缩）、EAPO（工具滥用抑制）、SISA（架构融合）
10. **代理评估过程化**：TEL-DRIFT（span级错误定位）、POIROT（代理自检）、AgentCL（持续学习评估）
11. **代理安全内生化**：SafeSteer（局部安全对齐）、HLL（CAPTCHA验证边界）
12. **SSM-Attention分数级融合**：SISA定义第三设计轴
13. **世界模型协同进化**：COMAP解决固定世界模型无法适应演化代理问题
14. **代理技能内部化**：SIRI技能内部化而非检索
15. **局部安全对齐**：SafeSteer 100样本实现 对齐成本-100x
16. **推理效率从"短轨迹"到"无冗余"**：ThoughtFold 内省式冗余识别
17. **代理评估从"结果"到"过程"**：NovelAPIBench/StepFinder/ChemCoTBench-V2
18. **形式化证明从"专用"到"通用+协作"**：LEAP 解决 Putnam 全部 12 题
19. **医疗 AI 从"问答"到"多轮决策+GUI 操作"**：ClinicalMC/MedCUA-Bench/CP-Agent
20. **基准审计可靠性缺口**：统计检测 59% 正确率
21. **Token 效率优化成为 Agent 经济性核心**：headroom (3530 stars/日) + 8v + DeepClaude 形成完整优化栈
22. **推理图拓扑分析**：ICML 2026 将推理转化为可测量图结构，超越 token 计数和准确率
23. **视觉锚定 RL**：VEPO 揭示视觉敏感 token 在 RLVR 中的关键作用
24. **金融代理基准**：Hedge-Bench 102 真实对冲基金任务，前沿 <16%
25. **Delta Debugging 引入 LLM安全测试**：DDOR 经典方法新应用
26. **多代理编排工程化**：SPOQ 17 仓库 8589 commits 纵向研究
27. **代理-仪器协议**：LAP 填补 MCP/A2A 之间物理世界空白
28. **确定性记忆框架**：DMF 零 token 记忆管理
