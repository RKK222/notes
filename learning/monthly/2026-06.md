## 本月技术趋势（前 2 日观察）

1. **Agent 评估基础设施化**：GLIDE（PPI 估计器统一库）和 TraceGraph（轨迹决策图谱）同时出现，标志着 Agent 评估从"手工跑基准"走向"标准化基础设施"
2. **专家知识蒸馏→Agent 技能包**：COLLEAGUE.SKILL（18.5K stars）代表"将人类 expertise 编码为 Agent 技能"的新范式
3. **形式化验证 × RL 融合加速**：Dafny RLVR、Value Function 即 Supermartingale 证书、ASP-based RL 抽象三条独立工作同时出现
4. **JEPA 架构扩展**：CHARM 将 JEPA 从视觉扩展到时间序列，ICML 2026
5. **Agent 推理结构显式化**：LinTree 揭示"搜索历史本身不够，必须显式表示树结构"
6. **凭证检测从二元到三分类**：CNN-CodeBERT（ICSME 2026）显著降低误报
7. **RLVR 小模型代码生成**：Qwen3-0.6B/Llama3.2-1B + RLVR pass@1 +13pp，奖励设计敏感性
8. **AI 基础设施投资加速**：Alphabet $80B 融资 + OpenAI on AWS
