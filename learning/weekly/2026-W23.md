## 本周主要研究方向

### 核心主线（AI Testing / Software Testing）
- **TraceGraph**：Agent 轨迹决策图谱，SWE-bench 恢复率 40.4%→43.5%，过程级诊断替代聚合分数
- **GLIDE**：PPI 评估基础设施统一库，ICML 2026 Workshop，解决标注成本与偏差 trade-off
- **COLLEAGUE.SKILL**：专家知识蒸馏→Agent 技能包，18.5K stars，可检查可纠正可部署
- **LLM-FACETS**：隐私保护 LLM 透明度评估框架，对齐 EU AI Act 合规需求
- **R+R**：Java 安全 API 滥用最新评估（GPT-5.5 + Llama-3.3-70B），RAG 知识效果因模型而异
- **形式化验证 + RLVR**：Dafny/Lean 统一框架，规格黑客问题，Dalek-Bench 仓库级基准
- **FASR**：STPA 安全分析自动化，航空电子系统 UCA 自动识别
- **逆向工程 Agent 安全**：二进制文件提示注入攻击检测与防御
- **RLVR 数据合成**：ADR 原子分解重组解决训练数据瓶颈

### AI / LLM 技术
- **AutoSci**：全科研生命周期记忆中心 Agent 系统
- **LinTree**：LLM 推理显式搜索历史树结构，父指针是关键
- **HypoAgent**：知识图谱交互式溯因假设生成，三 Agent 协作
- **SCALE**：Web Agent 认知感知自改进，SCALE-20k 数据集
- **CoSee**：多模态 Agent 共享状态协作失败模式审计，"更多计算≠更好性能"
- **GraphARC**：图抽象推理基准，KDD 2026，揭示理解-执行差距
- **VecLinking**：跨模型向量链接，ICML 2026
- **因果路径形式化**：罕见事件因果分析，ICML 2026
- **卫星集群多 Agent**：差分 Transformer，ECML-PKDD 2026

### 新领域探索
- **CHARM**：时间序列 JEPA，ICML 2026，通用时间序列表征
- **TxFM**：基因表达掩码自监督学习，"精心策划小数据集 > 大规模图谱"
- **Chem-PerturBridge**：37K 化合物扰动转录组资源
- **误差反馈紧理论**：分布式优化理论基础
- **Transformer 注意力头机制**：位置 vs 符号，RoPE 几何
- **超鞅证书 RL**：值函数即形式保证
- **ASP RL 抽象**：逻辑编程引入 RL，ICLP 2026
- **食物即药物推理**：FAM-Bench 多模态医疗 AI
- **上下文依赖论证**：CDAF 扩展 Dung 理论，FLoC 2026
- **工业 PLC 翻译**：LLM 梯形逻辑翻译
- **化合物扰动转录组**：药物发现基础设施
