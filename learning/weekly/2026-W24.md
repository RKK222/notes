## 本周主要研究方向

W24 聚焦 Agent 自我进化、AI 安全评估重构、形式化验证、Glassbox AI 架构、推理效率优化、Agent 技能生态、人类-Agent 协作协议、自主网页导航、RAG 推理加速等方向。累计扫描 7 次，覆盖 ~237+ 项技术内容，深度研究 34+ 项。

### 核心发现

1. **Agent 自我进化闭环**：Socratic-SWE 从求解轨迹蒸馏结构化技能，SWE-bench Verified 达 50.40%；OpenSkill 实现无监督开放世界技能构建；SkeMex 医疗 Agent 后部署自我进化；Anything2Skill 将外部知识编译为可复用技能。Agent 从"被动接收合成数据"进入"主动从自身经验学习"阶段。

2. **AI 安全评估重构**：Attack Selection 揭示策略性攻击使安全评估下降 20-28pp；PRIME 将奖励黑客研究从"事后检测"推进到"事前预警"；CHAP 填补 MCP/A2A 之间的人类-Agent 可问责协作空白；安全 Judge 课程训练实现跨评分标准稳健。

3. **推理效率从固定到自适应**：SIFT RAG 预填充加速 1.71x（存储比 KV 小 24000x）；EntropyInfer 长上下文自适应推理 2.39x 加速；光学推理图像作为推理媒介 1.96x token 效率；容量非格式结构化推理先推理后格式化策略。

4. **Agent 从对话到自主操作**：RunAgent SuperBrowser 基于人类浏览理论 89.47% Mind2Web Hard 成功率；AliyunConsoleAgent 云控制台 63.52% 成功率；自主事件解决超大规模网络 90%+ 自主解决率。

5. **技能生态标准化**：Anything2Skill 知识→技能编译；google/skills 12K stars；SkeMex 技能记忆管理。从知识检索推进到技能编译成为新范式。
