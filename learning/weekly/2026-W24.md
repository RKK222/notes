## 本周主要研究方向

W24 聚焦 Agent 自我进化、AI 安全评估重构、形式化验证、Glassbox AI 架构、推理效率优化、Agent 技能生态、人类-Agent 协作协议、自主网页导航、RAG 推理加速、Agent 记忆新范式、AI 辅助认知影响、商业世界模型、Agent 技能安全、基础设施感知编排、自改进 AI 闭环、环境工程自主研究、Agent 原生知识图谱、编排级奖励建模等方向。累计扫描 14 次，覆盖 ~460+ 项技术内容，深度研究 60+ 项。

### 核心发现

1. **Agent 自我进化闭环**：Socratic-SWE 从求解轨迹蒸馏结构化技能，SWE-bench Verified 达 50.40%；OpenSkill 实现无监督开放世界技能构建；SkeMex 医疗 Agent 后部署自我进化；Anything2Skill 将外部知识编译为可复用技能。Agent 从"被动接收合成数据"进入"主动从自身经验学习"阶段。

2. **AI 安全评估重构**：Attack Selection 揭示策略性攻击使安全评估下降 20-28pp；PRIME 将奖励黑客研究从"事后检测"推进到"事前预警"；CHAP 填补 MCP/A2A 之间的人类-Agent 可问责协作空白；安全 Judge 课程训练实现跨评分标准稳健。

3. **推理效率从固定到自适应**：SIFT RAG 预填充加速 1.71x（存储比 KV 小 24000x）；EntropyInfer 长上下文自适应推理 2.39x 加速；光学推理图像作为推理媒介 1.96x token 效率；容量非格式结构化推理先推理后格式化策略。

4. **Agent 从对话到自主操作**：RunAgent SuperBrowser 基于人类浏览理论 89.47% Mind2Web Hard 成功率；AliyunConsoleAgent 云控制台 63.52% 成功率；自主事件解决超大规模网络 90%+ 自主解决率。

5. **技能生态标准化**：Anything2Skill 知识→技能编译；google/skills 12K stars；SkeMex 技能记忆管理。从知识检索推进到技能编译成为新范式。

6. **Agent 记忆新范式**：Infini Memory（主题结构化文档）、OSL-MR（约束优化记忆保留）、Latent Memory（单 token 潜在表示）、ActiveMem（前额叶-海马体分布式架构）。记忆系统从"被动存储"向"主动管理+潜在空间+分布式架构"演进。

7. **AI 辅助认知影响**：Exploratory Responsiveness 理论揭示 AI 辅助可能同时增强或削弱人类探索能力；Predictive Assistance 几何框架揭示外源探索压缩导致迟滞和延迟恢复。

8. **商业世界模型**：BWM 将世界模型从物理世界扩展到商业/组织环境，支持自主决策和反事实推理。

9. **LLM Judge 评估差距**：RealMath-Eval 揭示合成数据训练的 Judge 在真实人类推理评估中存在显著差距（MSE 2.96 vs 1.17）。

10. **Agent 改进可审计性**：Regimes 通过事件源架构使 Agent 自主改进成为可追溯工作流。

11. **Agent 技能安全化**：SkillSpector 64 漏洞模式×16 类别扫描器，26.1% 技能含漏洞，Agent 安全从模型层向技能层扩展。

12. **自改进 AI 闭环**：SIA 框架 Meta/Target/Feedback 三 Agent循环，MLE-Bench #1，LawBench 70.1%（先前 SOTA 45%），Agent 从被动接收合成数据到主动自我改进。

13. **Agent 技能工程方法论**：agent-skills 24 个生产级技能覆盖完整 SDLC，Superpowers 子代理驱动开发+TDD+两阶段审查，Agent 从辅助编码到自主交付。

14. **基础设施感知编排**：INFRAMIND 将基础设施信号纳入多 Agent 编排决策，高负载下 99.9% SLO 合规（基线 <50%），从任务中心到基础设施中心范式转变。

15. **科学推理评估诚实化**：SciConBench 9.11K 问题最佳 Agent F1 仅 0.337，清洁室评估揭示数据泄露膨胀能力估计。
