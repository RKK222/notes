## 可能原因

1. **浏览器会话过期**：Browser 工具会话长时间未活动
2. **Gateway 进程问题**：浏览器控制服务可能卡死
3. **资源限制**：系统内存或 CPU 限制导致浏览器崩溃

---

## 解决方案

### 立即操作
```bash
# 重启 OpenClaw Gateway
openclaw gateway restart
```

### 验证步骤
1. 检查 Gateway 状态：`openclaw gateway status`
2. 验证浏览器功能：手动触发一次 Moltbook 访问
