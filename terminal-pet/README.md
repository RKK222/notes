# 🐾 TerminalPet - 终端电子宠物

在命令行里养一只属于你的电子宠物！

## ✨ 特性

- 🐶 多种宠物选择（猫/狗/龙/像素怪）
- ❤️ 完整状态系统（饥饿/心情/精力/清洁）
- 🎮 丰富互动（喂食/玩耍/睡觉/清洁）
- 📈 成长进化系统
- 💾 自动存档
- 🎨 终端 ASCII 艺术

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/RKK222/terminal-pet.git
cd terminal-pet

# 运行
python3 main.py
```

## 🎮 游戏命令

| 命令 | 功能 | 说明 |
|------|------|------|
| `feed` | 喂食 | 减少饥饿，增加体重 |
| `play` | 玩耍 | 增加心情，消耗精力 |
| `sleep` | 睡觉 | 恢复精力 |
| `clean` | 清洁 | 提升清洁度 |
| `status` | 状态 | 查看宠物详情 |
| `evolve` | 进化 | 查看进化条件 |
| `quit` | 退出 | 保存并退出 |

## 📊 状态说明

| 状态 | 说明 | 过低后果 |
|------|------|----------|
| 饥饿 | 0-100 | <20 会生病 |
| 心情 | 0-100 | <30 会抑郁 |
| 精力 | 0-100 | <10 会晕倒 |
| 清洁 | 0-100 | <30 会生病 |
